# ThanasOS (macOS themed Portfolio)

ThanasOS is a browser-based portfolio website designed to recreate the macOS desktop experience. Instead of a traditional portfolio layout, visitors interact with a desktop-like interface where they can open apps, move windows, and explore different sections such as About, Projects, and Journey.

This project was one of the first projects where I started seriously building websites and experimenting with UI design and interactions. While working on it, I learned a lot about structuring frontend projects, creating interactive components, and integrating different services into a single interface.

The goal of this project was to make a portfolio that feels more immersive and memorable than a standard webpage.

## Tech Stack

- React  
- TypeScript  
- Tailwind CSS  
- Vite  
- Framer Motion
